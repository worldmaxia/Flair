🔗 App Linkate
📖 Come pubblicare la tua app
▼
Trascini il file → link pronto in 30 secondi.
Nessun account richiesto per iniziare.
PASSO 1 — Vai su Netlify Drop
Apri drop.netlify.com dal browser
PASSO 2 — Trascina il file
Trascina il tuo .html nell area tratteggiata
oppure tocca Browse to upload
PASSO 3 — Copia il link
Netlify genera un link tipo:
amazing-name-123.netlify.app
PASSO 4 — Aggiungi qui
Torna → App Linkate → + Aggiungi → incolla il link
Ideale per app multi-file o quando vuoi
tenere la storia delle modifiche.
PASSO 1 — Crea account GitHub
Vai su github.com → Sign up → gratuito
PASSO 2 — Nuovo repository
+ New → nome (es. mia-app) → Public → Create
PASSO 3 — Carica il file
Add file → Upload → carica index.html → Commit
PASSO 4 — Attiva Pages
Settings → Pages → Branch: main → Save
Link pronto in ~2 minuti:
tuonome.github.io/mia-app
PASSO 5 — Aggiungi qui
Torna → App Linkate → + Aggiungi → incolla il link

📂 Files
⚠️ File System Access API non supportata su questo browser.
Usa Chrome su Android/PC per abilitare l'accesso ai file reali.
I link già salvati restano visibili ma non riapribili.
☁️ Come usare file da Dropbox / Drive
▼
Il file rimane sul cloud — qui nel launcher salviamo solo un collegamento.
La prossima volta che lo apri, viene letto direttamente dalla tua cartella cloud.
PERCHÉ È UTILE
✅ Modifichi il file su PC → si aggiorna automaticamente anche sul telefono
✅ Il launcher rimane leggero — nessun codice salvato dentro
✅ I file sono al sicuro anche se pulisci il browser
✅ Puoi aprirli da qualsiasi dispositivo con la stessa app cloud
COME FUNZIONA IL PICKER
Quando tocchi + Aggiungi, si apre il selettore file del telefono.
Lì trovi anche le cartelle Dropbox / Drive / OneDrive
(solo se l'app cloud è installata sul telefono).
Seleziona il file → il launcher lo registra e lo ricorda.
⚠️ Importante: Chrome chiede conferma permesso ad ogni sessione (ogni volta che riapri il browser). È un limite di sicurezza di Android, non aggirabile. Basta toccare Consenti quando compare.
Dropbox è il più affidabile per accedere ai file dal picker Android.
Consigliato per chi già lo usa o vuole sincronizzare su più dispositivi.
PASSO 1 — Installa Dropbox
Se non ce l'hai, installalo dal Play Store.
Crea un account gratuito (2GB inclusi).
PASSO 2 — Crea la tua cartella
Nell'app Dropbox: + → Crea cartella
Chiamala come vuoi, es. MieApp oppure Flare
PASSO 3 — Carica i file HTML
Dropbox app → entra nella cartella → + → Carica file
Seleziona i tuoi file .html dalla cartella Download.
PASSO 4 — Registrali qui
Torna nel Launcher → Files → + Aggiungi
Nel picker del telefono tocca Dropbox (voce nel menu laterale)
Naviga nella tua cartella → seleziona il file → fatto!
PASSO 5 — Aggiornare in futuro
Modifica il file sul PC → Dropbox sincronizza → sul telefono
la prossima apertura carica già la versione nuova. Zero operazioni extra.
⭐ Consigliato per Android
Google Drive è già integrato su Android — nessuna installazione extra.
15GB gratuiti, perfetto se hai già un account Google.
PASSO 1 — Crea cartella su Drive
Apri Google Drive → + → Nuova cartella
Chiamala es. MieApp
PASSO 2 — Carica i file
Entra nella cartella → + → Carica file
Seleziona i file .html dalla memoria del telefono.
PASSO 3 — Abilita accesso offline
Per ogni file: tieni premuto → Rendi disponibile offline
⚠️ Importante: senza questo il file non è accessibile senza internet.
PASSO 4 — Registra nel Launcher
Files → + Aggiungi → nel picker scegli Drive
Naviga → seleziona il file → registrato!
💡 Su Android, Drive appare automaticamente nel file picker di Chrome. Non serve fare nulla di extra.
⭐ Consigliato per PC Windows
OneDrive è la scelta ideale se usi già Microsoft 365 o un PC Windows.
5GB gratuiti, sincronizzazione automatica con la cartella Documenti di Windows.
PASSO 1 — Installa OneDrive
Se non ce l'hai → Play Store → Microsoft OneDrive
Accedi col tuo account Microsoft (gratuito).
PASSO 2 — Crea cartella
App OneDrive → + → Crea cartella
Oppure usa direttamente la cartella Documenti già presente.
PASSO 3 — Carica i file HTML
Entra nella cartella → icona upload → seleziona i tuoi .html
PASSO 4 — Registra nel Launcher
Files → + Aggiungi → nel picker scegli OneDrive
Seleziona il file → registrato!
VANTAGGIO SU PC WINDOWS
Su Windows, OneDrive sincronizza automaticamente la cartella
C:\Users\TuoNome\OneDrive\MieApp\
Salvi il file lì → in pochi secondi è aggiornato anche sul telefono.
